# MIYAIP 数据运营中心

本项目是 MIYAIP 数据运营中心的前端交互原型，包含数据可视化面板与运营管理模块

🔗 在线预览
- 部署地址：https://dcao1255-ctrl.github.io/miyaipdata/

✨ 功能特性
- 📊 数据大屏：实时展示运营数据与趋势图表
- 📱 响应式适配：支持 PC 端与移动端浏览
- ⚡ 轻量高效：基于 Vite 构建，加载速度极快

🛠️ 技术栈
- 框架/库： HTML5, CSS3, JavaScript (ES6+)
- 打包工具：Vite
- 部署服务：GitHub Pages

🚀 快速开始（针对开发者）
如果你想在本地运行这个项目：

1. 克隆项目到本地：
   git clone https://dcao1255-ctrl.github.io/miyaipdata/

2. 安装依赖并运行：
   npm install
   npm run dev
